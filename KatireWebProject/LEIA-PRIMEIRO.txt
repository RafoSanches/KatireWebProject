KATIRE - ENTREGA FINAL
======================

COMO ABRIR
----------
1. Extraia todo o ZIP.
2. Abra a pasta KatireWebProject.
3. Execute INICIAR-KATIRE.bat.
4. Aguarde e acesse http://localhost:4200.

Alternativa pelo terminal, dentro da pasta:

  npm install
  npm run dev

Aplicacao: http://localhost:4200
Banco JSON Server: http://localhost:3000
Dados persistidos: db.json

Para uma apresentacao rapida sem instalar, abra index.html.

LOGIN GOOGLE
------------
Para ativar o botao Google real, crie um Client ID OAuth no Google Cloud,
autorize http://localhost:4200 e informe o ID em google.config.json.
O login comum funciona sem essa configuracao.

IMPORTANTE
----------
O ZIP nao precisa de node_modules. O comando npm install recria as
dependencias automaticamente.
